# 夹心 Room -- 琴房预约系统 图片素材清单

> **品牌名**: 夹心 | **项目名**: 夹心 Room
> **风格方向**: Figma (活泼多彩) x Apple (极致留白精致) 融合
> **色调**: 主色 `#0071E3` (蓝)、辅色 `#34C759` (绿)、点缀 `#FF9500` (橙)、背景 `#FFFFFF` / `#F5F5F7`
> **插画风格**: 扁平 3D / 等距风格 (Isometric)，干净线条，大圆角，柔和渐变，统一光源（左上方 45°），带轻微阴影
> **人物风格**: 简约几何人物，无面部细节，活泼姿态，校园感
> **用途说明**: 用于高校琴房预约系统的 Web + 移动端 UI 界面

---

## 一、品牌 Logo (2 张)

### 1. `logo-main.svg`
- **尺寸**: 矢量 SVG，导出 512x512 PNG 备用
- **内容**: "夹心 Room" 品牌 Logo
- **要求**: 以钢琴键盘为核心元素，白键与黑键组成一个抽象的"夹"字形态（中间夹着一颗心/音符）。配色使用 `#0071E3` 蓝色为主。极简风格，去除多余装饰。需要在 24px 小尺寸下依然清晰可辨。
- **使用位置**: 网站 Header、登录页、管理后台侧边栏、PWA 图标、浏览器 Favicon

### 2. `logo-text.svg`
- **尺寸**: 矢量 SVG，高度 40px 标准
- **内容**: "夹心 Room" 品牌文字 Logo（中文"夹心" + 英文 "Room"）
- **要求**: "夹心"使用圆润的无衬线中文字体，"Room"使用现代感的英文字体（略细于中文部分形成层次）。"夹"字中间可巧妙融入音符元素。整体简洁现代，与 Apple 风格一致。纯色版本，颜色 `#1D1D1F`
- **使用位置**: Header Logo 旁、页脚品牌区

## 二、登录 / 注册 / 忘记密码页面 (3 张)

### 3. `auth-login-hero.png`
- **尺寸**: 1200x1600（竖版，用于登录页左侧面板）
- **内容**: 一个大学生坐在钢琴前练琴的等距插画场景
- **要求**: 场景包含一架三角钢琴、一位戴耳机的学生（侧面坐姿）、周围飘浮着音符气泡。房间有大窗户透进柔和阳光。整体暖色调，主色蓝 `#0071E3`，辅以浅橙 `#FF9500` 光晕。风格干净通透，Apple 产品宣传页既视感。
- **使用位置**: 登录页左侧大图区域，作为背景图使用（上层会叠加半透明毛玻璃效果）

### 4. `auth-register-hero.png`
- **尺寸**: 800x800（方形构图）
- **内容**: 多个学生在不同琴房练习的俯瞰等距插画
- **要求**: 展示 3-4 间小琴房的剖面图（类似建筑切面），每间里有不同姿态的学生在弹琴、调音、看谱。楼道中有一个学生拿着手机在预约。色彩明快多彩（每间房间用不同的柔和色调），整体保持 Figma 的活泼协作感。
- **使用位置**: 注册页顶部插画

### 5. `auth-forgot-hero.png`
- **尺寸**: 600x600
- **内容**: 一封带音符封印的邮件信封插画
- **要求**: 等距视角的 3D 信封，封印处是一个音符图标。信封旁边有一把金色钥匙和一个盾牌。整体蓝灰色调，表达「安全找回」的含义。简洁扁平风，不超过 5 种颜色。
- **使用位置**: 忘记密码页顶部

## 三、首页 (4 张)

### 6. `hero-illustration.png`
- **尺寸**: 900x700
- **内容**: 首页 Hero 区域主视觉插画
- **要求**: 一架优雅的三角钢琴（中心偏右），琴盖微开，从中飘出彩色音符、五线谱和光粒子。钢琴周围环绕着代表"预约"的日历图标、手机界面、时钟等科技元素。底部有微微的校园天际线剪影。色彩用蓝色为主基调配多彩音符。这是整个网站最重要的一张图。
- **使用位置**: 首页 Hero 区域右侧（与左侧文案 + CTA 按钮平衡）

### 7. `room-placeholder.png`
- **尺寸**: 600x400（16:10.67 比例）
- **内容**: 一间精致琴房的等距插画（通用占位图）
- **要求**: 包含一架立式钢琴、琴凳、谱架、墙上的隔音板、一盏暖色台灯。房间干净整洁，地板木质纹理。风格温馨舒适。色彩柔和，以米白和浅木色为主，钢琴为黑色。
- **使用位置**: 琴房列表卡片默认图（当琴房没有上传实景照片时的 fallback）、琴房详情页顶部

### 8. `step-icons.png`（或拆分为 4 张独立图）
- **尺寸**: 每个图标 200x200，共 4 个
- **内容**: 预约流程四步图标（统一风格）
  - **8a** `step-login.png`: 一个手机屏幕上显示登录界面 + 学生证
  - **8b** `step-search.png`: 一个放大镜聚焦在琴房列表上
  - **8c** `step-book.png`: 一只手点击"确认预约"按钮，按钮上方弹出绿色对勾
  - **8d** `step-play.png`: 一位学生坐在钢琴前开始弹奏，周围飘着音符
- **要求**: 四个图标风格必须完全统一（等距 3D 扁平）。主色 `#0071E3`，每个图标有一个辅助色（蓝/紫/绿/橙对应四步）。无描边纯色块风格。圆角矩形底座。
- **使用位置**: 首页"如何预约"流程区域（替代当前的 emoji 符号）

### 9. `piano-pattern.svg`
- **尺寸**: 矢量 SVG，可平铺
- **内容**: 极淡的钢琴键盘 + 音符无缝纹理图案
- **要求**: 由简化的钢琴键、音符、五线谱片段组成的 subtle pattern。颜色为 `#000000`，透明度 3-5%（极淡，几乎看不见但增加质感）。可无缝平铺。
- **使用位置**: Hero 区域背景纹理叠加、关于页面背景、管理后台 Dashboard 背景

## 四、空状态插画 (4 张)

### 10. `empty-no-rooms.png`
- **尺寸**: 400x300
- **内容**: 一间空荡荡的琴房，钢琴盖关闭，椅子推到一旁
- **要求**: 表达「暂无可用琴房」的含义。画面略显安静，使用灰蓝色调。可以有一个小音符在角落叹气的拟人化表达。简洁扁平风。
- **使用位置**: 琴房列表页筛选无结果时

### 11. `empty-no-reservations.png`
- **尺寸**: 400x300
- **内容**: 一本打开的空白日历/记事本 + 旁边一架小钢琴
- **要求**: 表达「暂无预约记录」。日历页面上没有任何标记，旁边有一支笔和一个"+"号气泡暗示"去创建"。温暖柔和的色调。
- **使用位置**: 我的预约页面无数据时、管理后台预约列表空状态

### 12. `empty-no-data.png`
- **尺寸**: 400x300
- **内容**: 通用空数据插画 -- 一个打开的空盒子 + 音符
- **要求**: 一个可爱的空纸箱，敞开着，里面飘出几个淡色音符。表达"这里还没有内容"。轻松友好的感觉。可复用于多个列表空状态。
- **使用位置**: 管理后台通用空状态（用户列表、日志列表等）

### 13. `404-not-found.png`
- **尺寸**: 600x500
- **内容**: 404 页面专用插画
- **要求**: 一架钢琴的琴键上，"404" 三个数字像琴键一样立起来。旁边有一个迷路的小音符角色（拟人化），手里拿着地图看得一脸困惑。背景简洁白色，色彩以蓝 `#0071E3` 和灰为主。整体有趣但不幼稚。
- **使用位置**: 404 Not Found 页面

每张图可以用以下格式发给 Gemini：

```
请生成一张 [扁平等距/3D扁平] 风格的插画：

主题：[上方"内容"字段]
尺寸比例：[宽x高]
具体要求：[上方"要求"字段]

风格参考：Apple 官网产品展示的干净简洁感 + Figma 品牌的活泼多彩感。
配色：主色 #0071E3，辅色 #34C759 / #FF9500 / #5AC8FA，背景白色或极浅灰 #F5F5F7。
不要文字水印。背景透明或纯白。
```
